Detailed Description

Interface of a graphic lib for the Core program.

This interface will be used by the Core program to manipulate a graphic lib without knowing any detail about it The Core will always do "request" to the lib, to get information or ask for displaying the map and GUI, for example. The IGfxLib will never have a reference to the Core, or communicate directly with it. Never. Ever.
